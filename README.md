# 📱 Weather Analytics Dashboard
## An Interactive Power BI Dashboard for Forecast Insights & Environmental Intelligence

A modern and visually compelling dashboard designed to monitor real-time weather conditions, analyze forecast trends, and evaluate air quality metrics—enabling smarter, data-driven environmental insights.

### 🚀 Overview
The Weather Analytics Dashboard provides a centralized view of key climate indicators such as temperature trends, air quality index (AQI), humidity, and precipitation. It transforms raw weather data into meaningful visual insights for better understanding and decision-making.

### 📌 Problem Statement
Weather and environmental data is often complex, scattered, and difficult to interpret quickly. Users face challenges like:
- Difficulty tracking temperature changes over time
- Lack of clarity in air pollution levels
- Inability to analyze multiple weather parameters together
- Poor visualization of forecast patterns

### 🎯 Objective
To design an interactive dashboard that:
-Monitors real-time weather conditions
-Visualizes temperature trends and forecasts
-Tracks air quality and pollutant distribution
-Predicts rainfall probability
-Compares weather across multiple cities

### 🛠️ Tech Stack
- Power BI Desktop – Data visualization & dashboard creation
- Power Query – Data cleaning & transformation
- DAX – KPI calculations and dynamic measures
- Data Modeling – Structured relationships across datasets
- File Formats – .pbix (development), .png (preview)

### 📂 Dataset Description
The dataset includes real-time and forecast weather data across cities, covering:
- 🌡️ Temperature (current & forecast)
- 🌫️ AQI Components (PM10, CO, NO₂, SO₂, O₃)
- 💧 Humidity & Atmospheric Pressure
- 🌬️ Wind Speed & Visibility
- ☀️ UV Index
- 🌅 Sunrise & Sunset Timings
- 🌧️ Precipitation & Rain Probability

### ✨ Key Features
- 🔄 Dynamic and interactive filtering
- 📊 Multi-layered data visualization
- 🌍 City-wise comparison
- 📈 Forecast trend analysis
- 🌫️ Air quality breakdown
- 🎯 KPI-focused dashboard design
